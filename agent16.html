Parallel Specialist Agents

Four channel agents fire simultaneously via Promise.all — Email, LinkedIn, Twitter/X, and Product Hunt — each with a distinct system prompt, persona, and channel-specific constraints. No sequential waiting.

🎯

Orchestrator-First Design

The orchestrator agent runs first, extracting the core message, audience, and tone before any specialist fires. Specialists receive a structured brief — not raw source text — keeping each context window lean and consistent.

✅

Synthesiser Quality Pass

A final synthesiser agent reviews all four outputs for tone, message, and factual consistency — flagging any contradictions and confirming the campaign pack is ready to publish across channels.
