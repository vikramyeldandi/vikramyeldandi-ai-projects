# Luminary Home Assistant — Agent 01

A customer service FAQ agent for a fictional homeware retailer, 
built with the Anthropic API.

## Setup

1. Clone this repo
2. Open `agent01_chat_v2.html` in a text editor
3. Replace `YOUR_ANTHROPIC_API_KEY_HERE` with your Anthropic API key
4. Open the file in a browser

## Get an API key
Sign up at console.anthropic.com → API Keys → Create Key

## Note
Never commit your API key to version control.
