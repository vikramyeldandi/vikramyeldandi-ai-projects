Live MCP Tool Calls

Each analysis triggers four real-time calls to the GitHub API via a Cloudflare-hosted MCP server — get_repo, get_readme, list_commits, and list_issues — with the tool trace visible in every response.

📊

Activity Signal

Commit velocity, star count, fork count, and open vs closed issue ratio are surfaced as structured metrics — giving a fast read on whether a project is active, slowing, or effectively abandoned.

🩺

Health Verdict

The briefing concludes with a plain-language verdict — Healthy, Moderate, or Stalled — backed by the data Claude pulled, not a generic score. Useful for due diligence, dependency evaluation, or onboarding.
